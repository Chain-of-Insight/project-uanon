# uanon-client

## Project setup
```
cd client
npm install -g @vue/cli
npm install
cp env.example .env
# Edit .env to your environment
```

### Compiles and hot-reloads for development
```
npm run develop
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
